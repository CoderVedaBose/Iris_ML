# Iris Flower Predictor

A polished web app for predicting Iris flower species from physical measurements using a Random Forest classifier.

## Features
- Interactive browser-based prediction form
- Modern hero section and responsive UI
- Dark/light theme toggle
- Animated result card with confidence details
- Ready for deployment on Render or similar platforms

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
3. Open http://localhost:5000

## Deployment
The app is ready to deploy as a standard Flask service on Render, Railway, or Heroku. A Procfile is included for deployment.
